# SportyStyle

## Autenticación

Se utiliza Auth0 para login seguro, sin manejo manual de JWT.

## Carrito

Se usa Session Storage para mantener productos durante la sesión.

## Flujo

1. Login
2. Agregar productos
3. Validación formulario
4. Pago simulado
5. Limpieza de sesión

## Seguridad

Session Storage evita persistencia después del cierre de sesión.
